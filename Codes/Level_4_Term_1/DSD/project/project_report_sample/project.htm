A Brief Description

In this Project we have designed a device that involves both an arduino based system and an android app used for identifying the comfortable and correct posture for an individual and to alarm the said user if wrong posture is detected.

When the wrong posture is detected a vibration motor will vibrate to let the user know about it physically and an alarm will be sent to his/her android device.





Required Equipments

  • Arduino

  • Flex Sensor

  • Vibrator Motor

  • Bluetooth Module

  • AC-DC adapter

  • Op-Amp





How it works?

In this project Arduino microcontroller is used for interfacing to various hardware peripherals. The current design is a combination of an android app and an arduino hardware circuit. It will alarm the user of the wrong posture detected through an alarm in the app and also a vibration generated by a vibration motor. For doing so the Arduino microcontroller is interfaced to a bluetooth module, a flex sensor and a vibration motor. The flex sensor is attached to the back of the user to identify the current posture of the user. It acts as a voltage divider that maps values from 0 to 1023 to voltages 0 to 5V. It is these mapped values that we use identify various forms of posture of the user. The bluetooth module is used to connect with the android app. This establishes a 2 way connection between the Arduino and the Android app. It sends the reading from the flex sensor to the android app and also sends signals from the android to the Arduino circuit. The vibration motor, connected to a digital output pin, will vibrate after wrong posture has been detected. We have introduced an Op-Amp to increase the sensitivity of the flex sensor. It is the output from this Op-Amp that gives the flex sensor reading to Arduino.
The flex sensor is attached to the user’s back. He/she turns on the android app. The app will detect the bluetooth device of the arduino circuit. The user will now calibrate by identifying his/her correct posture and recording it by the app.
If the reading from the flex sensor, amplified by the Op-Amp, crosses the recorded threshold for a certain amount of time, the vibrator motor will vibrate and an arduino circuit will let the app know that the threshold has been crossed for some time. Getting this signal the app will also set off an alarm.

Algorithm



       1.Start
              2.Power the arduino of the “Posture Corrector” system.
              3.Start the app
              4.Connect to the HC-05 bluetooth module.
              5.Fix the perfect sitting posture
              6.Press the “Fix Threshold” button to send message to arduino through bluetooth
              7.Arduino gets the message and fixes the threshold based on the current analog value voltage from flex sensor
              8.Arduino initializes a count variable for counting the number of times the flex sensor value has exceeded the threshold
              9.Arduino measures the flex sensor value
              10.If value>threshold, then count=count+1
              11.If value< threshold, then count=count-3
              12.If count>40, then send alarm to mobile through bluetooth and vibrate the the motor. Then go to step 8
              13.If count< 40 then go to step 8
       14.End

Problems Faced

  1. Flex sensor calibration for individual users was problematic due to its lower sensitivity. Op-Amp was used to increase the sensitivity

  2. Bluetooth module was not initially working. So we had to develop a separate dummy app to test whether the bluetooth module correctly sent and received data.
